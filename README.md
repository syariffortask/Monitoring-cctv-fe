# Dashboard Monitoring CCTV
Frontend dashboard menggunakan svelte dan leafleat js untuk map dan kordinat nya
## Developing
```
npm install
```
## Copy env
sesuaikan url api dan WS api
```
cp .env.example .env
```

## Jalankan Dev Mode
```
npm run dev
```

## Building

```
npm run build
```

## Building image docker

```
docker run build -t fe-monitoring-cctv
```
